# EasyDocs Guide

A bilingual web application helping citizens apply for Passport, Voter ID, Birth Certificate, PAN Card, and Driving License.

## SDG Goal
SDG 16.9 - Legal Identity for All

## Features
- 5 document guides
- English + Hindi language
- Dark mode
- Interactive checklist with progress bar
- PDF download
